# Netflix-Data-Analysis-using-Pandas
Netflix data analytics dashboard using Pandas and Matplotlib

This project analyzes Netflix Movies and TV Shows data using Python, Pandas, and Matplotlib.

The project focuses on:
- Data Cleaning
- Data Analysis
- Visualization
- Content Insights

---

# Features

- Movies vs TV Shows Analysis
- Top Countries Producing Content
- Ratings Analysis
- Netflix Content Growth Analysis
- Data Visualizations using Matplotlib

---

#  Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

#  Dataset

Dataset used:
https://www.kaggle.com/datasets/shivamb/netflix-shows

---

#  Visualizations Included

- Line Chart
- Pie Chart
- Bar Chart

---

#  Key Insights

- Netflix content grew rapidly after 2015.
- Movies dominate the platform compared to TV Shows.
- The United States contributes the highest amount of content.
- Mature audience content is highly common on Netflix.
